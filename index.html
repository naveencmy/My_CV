Exam Hall Seat Allocation

Node.js Status: Private

The Architecture

Designed a deterministic algorithm to handle seat allocation for 1000+ students. The system parses diverse input constraints (subject codes, room capacity, student cohorts) and produces a collision-free map.

Engineering Trade-offs

Prioritized accuracy over speed in the initial compute phase, ensuring zero conflicts. The database schema was normalized to reduce redundancy, utilizing heavy SQL joins for the final report generation.
